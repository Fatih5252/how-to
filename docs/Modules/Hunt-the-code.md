---
title: Hunt the Code
---

Hunt the Code is a game where you have to find hidden codes in your Discord server. The goal is to find as many code snippets as possible. The server Moderators / Owner has to create the codes by himself and hide the codes by themself. The bot will not create or hide the codes for you. you can create codes with `/hunt-the-code-admin create-code` and can end the game with `/hunt-the-code-admin end`. the users then can redeem the codes with `/hunt-the-code redeem`.

Here, you can customize the embed message that will be used for the leaderboard. This embed will be sent when a user uses the `/hunt-the-code leaderboard` command. You can change the title, color, thumbnail and image. Please note that the thumbnail and image are optional. you don't have to set them. it could be empty.

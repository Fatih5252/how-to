# Replies

On this page, you can add to a user, if that user gets pinged, the bot will sends a message in the channel. you can customize the message here.

---

**User**
![image](/img/modules/automatic-reactions/user.png)

type: [String](https://howto.scnx-tutorials.de/type/string)

required: true

Here, you have to provide the user ID of the user you want to add a reply to. I know there’s no mention of a user ID, but please don’t provide the username only the user ID works here.

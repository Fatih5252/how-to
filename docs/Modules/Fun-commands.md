# Fun-commands
With this Module, you get access to fun commands such as 8ball, coinflip, dice, Number, random Ikea names, kiss, hug, slap and pat. You can use these commands to entertain your server members and make your server more lively.

---

**Hug images**
![image](/img/modules/fun-commands/hug-images.png)

type: [Add field](https://howto.scnx-tutorials.de/type/add-field)

required: false

Here, you can add custom hug images that will be used when a user uses the hug command. You can add multiple images here, and the bot will randomly select one when the command is used. If you don't add any images here, the bot will use its 3 default hug images.

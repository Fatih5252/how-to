# Messages

:::info
The Options **"leaderboardembed"**, **"liveleaderboardembed"** and **"embed"** are from the old times where slash commands didn't existed yet, Nothing happens if you fill in these options, you can leave them empty, but if you want to fill them in, then you can, but please be aware that these options are not used anymore, so they won't have any effect on the module.
:::

---
title: Invite Tracking
---

The invite tracking is a module that allows you to track the invites of your server. You can see who invited who, how many invites they have, and other information about the invites. This module is very useful for server owners who want to keep track of their invites and see who is inviting new members to their server. you can view the informations with the command `/trace-invites`. you can also select a channel where the bot will send a message when a new member joins the server, and in that message, you can see who invited the new member.

---

**Invite-Log-Channel**
![image](/img/modules/invite-tracking/invite.png)

type: [Channel Select](https://howto.scnx-tutorials.de/type/channel)

required: false

Here, you can select a channel where the bot will send a message when a new member joins the server. In that message, you can see the following informations: who joined, the invite link, in which channel the invite was created, which time the invite was created, which time the invite expires and from whom the invite was created. If you don't select a channel here, the bot won't send any messages when a new member joins the server.

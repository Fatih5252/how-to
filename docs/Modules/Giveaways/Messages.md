Here, you can edit all the messages that the Giveaways Module sends. You can customize them as you want.

---

**Button-Content**
![image](/img/modules/giveaways/button-content.png)

type: [String](https://howto.scnx-tutorials.de/type/string)

required: true

Here, you can customize the content of the button that users have to click to join the giveaway. This is the text that will appear on the button.

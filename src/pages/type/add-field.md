# Add field

Are you wondering what the "Type" section on this website means?  

If not, here is an explanation:

If you have to add a Field in SCNX, then it means that you can add multiple types of fields. you can add Fields as you want, its all your choice. 
For example, in the [Application module](https://howto.scnx-tutorials/docs/modules/applications) (section: Questions), you have to add multiple fields that contain questions that the users have to answer when they apply. 

you can add as many fields as you want by clicking the **"Add Field"** button.

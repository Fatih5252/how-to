# Add Element

Are you wondering what the "Type" section on this website means?

If not, here is an explanation:

This type means that you have to add elements and in that elements you have to do some configurations for example: select a channel, role or user or a custom message (this depends on the module you are using). You can add as many elements as you want, just make sure to configure them properly and please note that there is a Discord API limit.
For example, in the [Automatic Reactions module](https://howto.scnx-tutorials.de/docs/modules/automatic-reactions) (section: Everything except "Only react to @mentions?"), you have to add elements where you can select a channel and the emojis that the bot should react with when someone sends a message in that channel.

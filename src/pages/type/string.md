# String

Are you wondering what the "Type" section on this website means?

If not, here is an explanation:

The **String** type is similar to the [Spinner](https://howto.scnx-tutorials.de/type/spinner) type, but instead of only numbers, you can type anything letters, numbers, or symbols.

Depending on the module, you may be required to enter something in the field. Whether it’s required or optional depends on the specific module.

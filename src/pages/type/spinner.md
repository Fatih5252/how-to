# Spinner  

Are you wondering what the "Type" section on this website means?  

If not, here is an explanation:  

The **"Spinner" type** (also called *Number Input*) is a field where you can only enter numbers.  
That means you cannot type letters, symbols, or any other characters — only valid numerical values are accepted.  

To make things easier, the input also includes small **up and down arrows**. With these, you can increase or decrease the number step by step without having to type manually. This is especially helpful if you want to adjust values quickly.

# Embed

Are you wondering what the "Type" section on this website means?

If not, here is an explanation:

This type means that you have to create an embed message. An embed message is a special type of message that can contain a title, description, fields, and other elements. You can customize the embed message to your liking and use it in various modules. For example, in the [Hunt the Code](https://howto.scnx-tutorials.de/docs/modules/hunt-the-code) (section: Leaderboard-Message), you can create an embed message to display the leaderboard.
